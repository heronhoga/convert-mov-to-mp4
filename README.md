## this program could convert .mov video to .mp4
1. you need to install [ffmpeg](https://www.ffmpeg.org/download.html) first
2. clone this project
3. create "input" folder/directory which contains your pre-converted videos
4. run the program
5. check your coverted videos in the "output" folder/directory. if the program ran smoothly, all your videos should've been converted successfully
